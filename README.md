# Dictionary App

Welcome to the Dictionary App! This app is a powerful and user-friendly tool designed to help you look up words, listen to their pronunciations, and customize your experience with various settings. Built with React, Sass, and TypeScript, the app offers smooth performance and a modern interface.

Check out the live app [here](https://dictionary-app-aosman.netlify.app/).

## Features

### Word Search
- **Search Bar**: Quickly search for any word using the search bar at the top of the app.
- **Word Details**: View definitions, synonyms, antonyms, and example sentences for each word.
- **API Integration**: Fetches real-time data from a dictionary API to provide accurate and up-to-date information.

### Dark Mode
- **Toggle Dark Mode**: Switch between light and dark themes to suit your preference and reduce eye strain.

### Audio Pronunciations
- **Play Audio**: Listen to the correct pronunciation of words with a simple click.

### Font Customization
- **Change Font**: Choose from a variety of fonts to personalize the reading experience.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Sass**: A preprocessor scripting language that is interpreted or compiled into CSS.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.



## API Documentation

This app uses the [Dictionary API](https://dictionaryapi.dev/) to fetch word data. Refer to the API documentation for more details on endpoints and usage.
